# 🧮 Python Calculator

A simple command-line calculator built using Python. This program allows the user to perform basic arithmetic operations on multiple numbers.

## 📌 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* Accepts multiple input values
* Handles invalid menu selections
* Prevents division by zero

## 🚀 How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python calculator.py
```

## 📖 Example

```
ENTER 1 FOR ADDITION, 2 FOR SUBTRACTION, 3 FOR MULTIPLICATION, 4 FOR DIVISION:
1

YOU HAVE SELECTED ADDITION

ENTER THE NUMBER OF VALUES TO BE CALCULATED:

10
```

## ⚠️ Limitations

* Supports only integer inputs.
* Division operation currently follows the logic implemented in the code and may not behave like a standard calculator when dividing multiple numbers.
* Invalid numeric inputs are not handled using exception handling.
